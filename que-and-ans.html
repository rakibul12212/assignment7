What is the difference between map,filter,forEach and find function ?

  1. The map() method loops through each element in array and calls the provided function for each element. This method creates a new array and doesn't alter the original array.
  2. The filter() method in JavaScript creates a new array with the elements that satisfies the provided condition. This method calls a provided function for each element in array and verifies the condition given in the provided function and passes only those elements that satisfy the given condition.
  3. forEach() is used to execute the same code on every element in an array but does not change the array and it returns undefined.

What is the difference between local storage,session storage and cookies?

  1. Local storage is a feature in web browsers that allows a website to store data on a user's device. The data is stored persistently, meaning that it will still be available even after the user closes the browser. The data is associated with a specific domain, and can only be accessed by the website that stored it. Local storage is typically used for storing larger amounts of data that need to persist across multiple visits to a website.
  2. Session storage is similar to local storage, but the data is only available for the duration of the browser session. This means that the data is lost when the user closes the browser. Like local storage, session storage is associated with a specific domain and can only be accessed by the website that stored it. Session storage is typically used for storing temporary data that doesn't need to persist across multiple visits to a website.
  3. Cookies are small text files that are stored on a user's device by a website. They are used to store information about the user, such as their preferences or login credentials.
